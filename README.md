## Kedy.Cqrs.Template

This is a template for a CQRS project. Using .NET 8, Entity Framework, MediatR and xUnit. It is a simple project that can be used as a starting point for a new project. It is not intended to be a complete solution, but rather a starting point.

### Technologies

- [.NET 8](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)
- [Entity Framework](https://learn.microsoft.com/en-us/ef/core/)
- [MediatR](https://github.com/jbogard/MediatR/wiki)
- [Redis](https://redis.io/)
- [FluentValidation](https://fluentvalidation.net/)
- [Mapster](https://github.com/MapsterMapper/Mapster)
- [Serilog](https://serilog.net/)
- [Swagger](https://swagger.io/)
- [xUnit](https://xunit.net/)
- [Moq](https://github.com/devlooped/moq)
